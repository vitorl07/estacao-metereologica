# estacao-metereologica
Uma Simples Estação Metereológica usando 2 sensores e Arduino


## Esse projeto teve objetivo de demonstrar que é possivel obter dados metereológicos relevantes com baixo custo.

### Componentes

1 Sensor de Temperatura e Umidade DHT11
1 Sensor de Altitude, Temperatura e Pressão Atmosferica BMP280
1 Botão
1 Display Oled 0.96"
1 Arduino Uno
Varios Jumpers Macho-Femêa e Macho-Macho

### Sobre o código e a esquemática do circuito está em detalhes no pdf. Foi apresentado diretamente como um projeto de conclusão de disciplina na IFPB.
